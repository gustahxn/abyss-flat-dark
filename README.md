# Abyss Flat Dark

### EN
A VS Code theme that combines the **Default Dark Modern** interface with syntax colors inspired by **Theme Darker**. This version was customized to personal preference, featuring higher color saturation and refined background tones.

### PT-BR
Um tema para VS Code que combina a interface do **Default Dark Modern** com cores de sintaxe inspiradas no **Theme Darker**. Esta versão foi adaptada para gosto pessoal, com maior saturação de cores e tons de fundo refinados.

---

## Installation / Instalação

**Manual (.vsix):**
1. Open **VS Code**.
2. Go to **Extensions** (`Ctrl+Shift+X`).
3. Click the **"..."** (top right) and select **Install from VSIX...**.
4. Select the `abyss-flat-dark-0.0.1.vsix` file.

---

## Technical Details / Detalhes Técnicos

* **UI Base:** Dark Modern.
* **Syntax:** Inspired by Theme Darker (customized).
* **Editor Background:** `#1F1F1F`.
* **Terminal Background:** `#181818`.
