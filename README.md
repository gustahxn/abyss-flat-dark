# Abyss Flat Dark

### EN
A VS Code theme that combines the **Default Dark Modern** interface with syntax colors inspired by **Theme Darker**. This version was customized to personal preference, featuring higher color saturation and refined background tones.

### PT-BR
Um tema para VS Code que combina a interface do **Default Dark Modern** com cores de sintaxe inspiradas no **Theme Darker**. Esta versão foi adaptada para gosto pessoal, com maior saturação de cores e tons de fundo refinados.

---

## Installation / Instalação

### EN
1. Download the latest `.vsix` file from the [Releases](https://github.com/gustahxn/abyss-flat-dark/releases) page.
2. In VS Code, open the **Extensions** panel (`Ctrl+Shift+X`).
3. Click the **"..."** (top right) and select **Install from VSIX...**.
4. Select the downloaded file and activate the theme using `Ctrl+K` then `Ctrl+T`.

### PT-BR
1. Baixe o arquivo `.vsix` mais recente na página de [Releases](https://github.com/gustahxn/abyss-flat-dark/releases).
2. No VS Code, abra o painel de **Extensões** (`Ctrl+Shift+X`).
3. Clique nos **"..."** (canto superior direito) e selecione **Instalar de VSIX...**.
4. Selecione o arquivo baixado e ative o tema usando `Ctrl+K` seguido de `Ctrl+T`.
